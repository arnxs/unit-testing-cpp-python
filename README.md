# Unit Testing C++ / Python

## Prerequisite

- Python 3.6.7
  - recommend using pyenv if possible
  - fswatch

## Setup

``` bash
# Install python project requirements
$ pip install -r requirements.txt
```

## Instructions

### Python TDD

``` bash
# activate Python TDD script
$ ./go-py.sh
```

### C++ TDD

``` bash
# activate C++ TDD script
$ ./go-cpp.sh
```
