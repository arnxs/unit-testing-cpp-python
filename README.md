# Unit Testing C++ / Python

## Prerequisite

- Python 3.6.x
- fswatch

## Setup

``` bash
# Install python project requirements
$ pip install -r requirements.txt
```

## Instructions

### Python TDD

``` bash
# activate Python TDD script
$ ./py-tdd.sh
```

### C++ TDD

``` bash
# activate C++ TDD script
$ ./cpp-tdd.sh
```
